# CSS06_text

| 속성 | 설명 |
| --- | --- |
| font-size | 글자 크기를 지정 |
| font-family | 글꼴 지정 |
| font-style | 글자의 스타일(기울기 등) 지정 |
| font-weight | 글자의 두께 지정 |
| text-align | 글자 정렬 지정 |
| line-height | 글자의 높이 지정
(수직 중앙 정렬에 이용, 박스의 높이와 같은 크기로 지정) |
| text-decoration | 글자에 다양한 효과 적용(none, 밑줄, 대쉬, 도트 등) |
| letter-spacing | 글자의 간격을 조정 |
| text-indent | 들여쓰기, 내어쓰기를 설정 |
